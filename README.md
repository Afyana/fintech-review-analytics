## 📌 Project Overview

This project analyzes Google Play Store reviews for three major Ethiopian banks:
- **Commercial Bank of Ethiopia (CBE)**
- **Bank of Abyssinia (BOA)** 
- **Dashen Bank**

The goal is to transform unstructured user feedback into actionable insights for product teams, identifying satisfaction drivers, pain points, and feature requests.

## 🎯 Business Challenge

Mobile banking adoption in Ethiopia is accelerating rapidly. User reviews contain valuable signals about product quality, but left unanalyzed, this feedback remains noise. This project builds an analytics pipeline to:

1. **Collect** - Scrape reviews from Google Play Store
2. **Analyze** - Classify sentiment and extract themes using NLP
3 **Store** - Design PostgreSQL database for production use
4. **Recommend** - Generate bank-specific product improvements
